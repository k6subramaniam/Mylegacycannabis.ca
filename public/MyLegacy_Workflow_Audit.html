Built vs Pending Steps

8 of 8 core steps complete
  • ✓
    OTP email send — 6-digit code via Resend API, 10-min expiry
  • ✓
    OTP SMS send — Twilio integration coded (sends if configured)
  • ✓
    OTP verify + login/register — Validates code, creates user with 25pt bonus, sets session
  • ✓
    Google OAuth — Full redirect + callback + profile completion flow
  • ✓
    Account lock/unlock — Admin can lock users, login blocked with message
⚠ Pending Items:
• Twilio not configured — All 3 env vars empty. SMS OTP silently fails.
• No rate limiting on /api/auth/send-otp — Bot can spam Resend email quota (100/day free tier).
• Google OAuth callback URL — Must add production domain in Google Cloud Console.
• JWT secret is hardcoded — Must rotate to random 64+ char string.
Use Cases: New customer registration (email OTP), returning customer login (email/Google), guest checkout (no auth required but ID needed), admin account management (lock/unlock), phone-first users (SMS OTP when Twilio enabled).
Business Impact: Auth is the gateway to the entire system. Without SMS, phone-only users can't register. Without rate limiting, a bot can drain your Resend quota. The hardcoded JWT secret means anyone who's seen the code can forge sessions.
Relevance: Critical path — every single purchase requires authentication or guest checkout. Age gate compliance is legally mandatory in Canada (Cannabis Act, 19+ in most provinces). Authentication directly impacts conversion rate.

Relevance: Cannabis is a repeat-purchase business. Loyalty programs increase customer lifetime value by 20-30%. The broken auto-earn is a trust destroyer. Referrals drive organic growth at near-zero cost.

What AI Extracts Per Product

7 fields
Category (Indica/Sativa/Hybrid/Ounce Deals/Shake), grade (AAAA/AAA+/AAA/AA/SHAKE), strain name, THC %, isNew flag, and prices for 5 weight tiers: 1g, 3.5g, 7g, 14g, 28g. Creates one product row per weight tier (e.g., "Pink Taco — 3.5g", "Pink Taco — 28g").
⚠ Pending Items:
• Flower category only — Won't parse edibles, vapes, concentrates, accessories.
• No product images — Imported products have no images. Shows Unsplash fallback.
• No rate limiting — Each parse = ~$0.01-0.03 OpenAI cost. No abuse protection.
• No CSV/spreadsheet import — Image-based only.
Use Cases: Admin receives new supplier menu (usually a JPG/PDF photo) → uploads to platform → AI extracts all product data → admin reviews, adjusts stock levels → one-click import creates/updates 165+ product rows. Replaces ~2 hours of manual data entry.
Business Impact: Key differentiator for operational efficiency. 33 strains × 5 tiers = 165 rows in 30 seconds vs 2 hours manual entry. Reduces data entry errors. Flower-only limitation means other categories still need manual entry.
Relevance: Cannabis suppliers update menus weekly. This tool makes menu updates a 1-minute task instead of a 2-hour task. Operational efficiency directly impacts profitability for a small team.
