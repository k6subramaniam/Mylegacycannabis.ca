## 2026-04-15 - Remove Invalid Hreflang Tags
**Learning:** Setting an hreflang for a locale that doesn't actually exist (like `fr-CA` when no French content is available) is a technical SEO error that misinforms search engines about available language variants.
**Action:** When configuring hreflang tags, ensure only supported locales and `x-default` are present to maintain accurate SEO mappings.
